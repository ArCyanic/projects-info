# 大气污染可视化
本项目我第一次参与的可视化项目，用于参加China Vis 2021挑战赛。
主要用到的技术有：JavaScript, D3, Echarts, html, css。
这算是我的启蒙项目，我从中学到的知识不少：
- HTML + CSS + JavaScript设计网页
- 设计多图表联动
- 处理数据量较大和图表较多时产生的异步问题
- 用D3设计相对复杂的多变量图表
- 成员合作等。
